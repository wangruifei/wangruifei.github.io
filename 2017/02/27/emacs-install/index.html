Emacs-Install

tool
  • emacs

1月份买了一个Macbook pro，本打算写一些简单的c++程序，但是自带的xcode真心大，好几个GB，而且安装出问题，弄的好不纯粹。于是开始入手emacs.
为什么不用vim，是因为我只要写c++程序，emacs自带了很多我需要的功能，比如编译和高亮显示代码，虽然这个vim也可以。但是emacs自带就有这些功能也不需要去手动调整配置，我属于懒汉。

Emacs 安装

由于emacs有terminal和GUI版本，一个是在命令行，一个是在图形界面，但是这两个其实没什么差别。因为在终端里展示的emacs和GUI的emacs布局和功能完全一样。因此你可以选择你需要的安装。我只安装了terminal下的emacs。同时用到了一个大神的emacs配置文件，因为原生的emacs不是和人性化，而这个大神的自定义配置非常好用。拿来主义主义，何乐不为？

GUI版本下载地址：https://emacsformacosx.com/download/emacs-builds/Emacs-25.1-1-universal.dmg

Mac键盘 M-x 按键调整

emacs需要经常使用到ctrl键和alt键，但是在mac键盘的默认配置中是没有alt按键的。此时需要对自己的键盘做调整。

我是将左边的option按键当成alt按键
1.在terminal模式下，找到emacs的配置文件.emacs.d/init.el 加如下配置更改键盘

(setq ns-right-alternate-modifier nil)

2.或者在term终端工具的“偏好设置”-“描述文件”-“键盘”，勾选“将option键作为Meta键”也可以

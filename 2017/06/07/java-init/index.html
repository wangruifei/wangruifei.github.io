learning
  • java编程思想

Java的初始化包括：成员初始化，构造器初始化，数组初始化

成员初始化

Java尽力保证：所有变量在使用前都能得到恰当的初始化，对于java类成员变量，在未指定值时会赋予默认值；而对于java方法的局部变量，java以编译时错误的形式来保证必须进行方法成员的初始化工作，用以提醒开发者。

构造器初始化

可以通过构造函数，进行一些成员数据的初始化工作；但是不能阻止自动初始化的进行，因为自动初始化工作在构造器被调用前就发生了。

初始化顺序

在类的内部，变量定义的先后顺序决定初始化顺序。并且变量初始化工作一定会在方法被调用前执行。即便变量定义是散落在各个方法之间。

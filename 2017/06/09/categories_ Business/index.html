Guide to Woodome VPN

Business
  • VPN


On this page, you will find the usages about Woodome VPN.

Introduction

Woodome VPN is the best one in China which is fast, stable, cheap and safety. It will encrypts your data by AES-256-CFB

Supported systems

  • IOS（iPad,iPhone）
  • MacOS(Macbook,Macbook Pro,Mac Air,iMac)
  • Windows(Windows PC)
  • Android(Android phone,Android pad)

How to use it

Download VPN App

1.Click the link —> Download to download app,will show this page

Note: Please use the target device ,which will run VPN,to click download
(for example: if you need to use VPN on iphone, you need to use iPhone to open download page.)
2.After Installing.
On android, the app is “shadowsocks”

On iPhone,iPad,the app is “Wingy”

On Mac, the app is “shadowsocksX”

On Windows, the app is “shadowsocks”

Get your VPN Account

2.Visit woodome.com

Login

after login you will get this page

this is an example configuration

Setting

  • Setting on iPhone,iPad
  • Setting on Windows
  • Setting on Android
  • Setting on Mac

Setting on iPhone

1.Open “Wingy” and cilck “+“ icon

2.Choose “shadowsocks”

3.Input your own VPN configuration to Wingy

4.Click “done” to save it

5.Click “Connect” to use VPN

6.All done, have fun!
try to visit www.google.com

Setting on Windows

1.Double click the “shadowsocks.exe” to run it, and find it of your task bar.

2.Right click shadowsocks, choose server -> edit server to add your server

3.Input your own VPN configuration to it,the click “ok”

4.Choose your server to use, as shown below

5.Enable System Proxy and Start on Boot

6.All done, have fun!
try to visit www.google.com

Setting on Android

1.Input your own VPN configuration to shadowsocks

2.Click “right top button” to run VPN

3.All done, have fun!
try to visit www.google.com

Setting on Mac

1.Run “shadowsocksX” on your Mac, you will find the icon appears on your top task bar

2.Right click the icon and click “server” -> “open server preferences”

3.Input your own VPN configuration to it

After inputting your VPN configuration ,then click “OK”
4.Choose your “server” to use, as shown below

5.Choose “auto proxy mode” and “trun shadowsocks on”

6.All done, have fun!
try to visit www.google.com
